JSON Formatter Demo

下面展示一组示例输入与输出，帮助快速理解本工具的使用方式。

示例输入
{"name":"Alice","skills":["JavaScript","Python"],"active":true,"score":98}
格式化输出
{
    "name": "Alice",
    "skills": [
        "JavaScript",
        "Python"
    ],
    "active": true,
    "score": 98
}

使用步骤

  1. 打开 index.html
  2. 将原始 JSON 粘贴到输入区
  3. 点击“格式化”或“压缩”按钮
  4. 查看输出结果并可一键复制

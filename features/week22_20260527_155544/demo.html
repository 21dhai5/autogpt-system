JSON 格式化器

本周任务选择了 Web 实用工具 方向，实现一个可直接运行的 JSON 格式化器。 该工具支持 JSON 格式化、压缩、语法校验和复制结果，适合作为前端工具类项目的基础模板。

使用说明

  • 点击 格式化：按所选缩进层级输出可读 JSON。
  • 点击 压缩：去除多余空白，生成单行 JSON。
  • 点击 校验：只检查 JSON 是否合法，不改变原文。
  • 点击 复制结果：复制右侧输出区域内容。
  • 启动项目：在当前目录执行 npm start，浏览器打开 http://127.0.0.1:3000。
